# Custom-Transformer-based-Time-Series-Anomaly-Detector

This project implements a Transformer encoder model to detect anomalies
in univariate time series data using reconstruction error.

## Technologies
- Python
- PyTorch
- Pandas
- scikit-learn

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Open `notebook/anomaly_detector.ipynb`
3. Run all cells

## Output
- Trained model saved in `checkpoints/`
- Performance metrics in `report/`
